# MyRPG

In this project, you are asked to make a small video game based on the rules of a [Tower Defense](https://en.wikipedia.org/wiki/Tower_defense).
***
## How to play ?
### Installation CSFML
```bash
./build_csfml.sh
```
### Build
```bash
make
```
### Usage
```bash
./my_rpg
```
